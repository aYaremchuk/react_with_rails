Try Rails with React
====================

This project was create to try use ReactJS with Ruby on Rails

### Basic project specs:
- Ruby 2.4.0
- Ruby on Rails 5.0.1

### DB adapter:
- SQLite

### Setup
* bundle
* copy config/database.example.yml to config/database.yml
* rake db:create
* rake db:migrate
* rake db:seed
* rails s
